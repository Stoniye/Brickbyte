# Getting Started

This guide will help you set up your local development environment to compile and run Brickbyte.

## Prerequisites

Brickbyte is written in Rust and relies on OpenGL. You will need:

* The latest stable [Rust toolchain](https://rustup.rs/) (`cargo`, `rustc`).
* A GPU and drivers that support **OpenGL 3.3 Core Profile** (or higher).

## Building and Running

Clone the repository and run the project using Cargo:

```bash
git clone https://gitlab.com/Stoniye/brickbyte.git
cd brickbyte
cargo run --release
```
